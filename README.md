## Hot Reload with Air

```sh
go install github.com/cosmtrek/air@latest


# init. create air.toml
# example: https://github.com/cosmtrek/air/blob/master/air_example.toml
# change target file = `cmd = "go build -o ./tmp/main.exe {path/to}targetfile.go"`
air init

# need go.mod
go mod init example
```
